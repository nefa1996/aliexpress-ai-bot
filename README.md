# AliExpress AI Telegram Bot

Этот бот автоматически публикует лучшие товары с AliExpress в Telegram-канал или чат.

## Установка

1. Создайте Telegram-бота через @BotFather
2. Получите API ключ AliExpress Partner
3. Создайте GitHub репозиторий и загрузите файлы

## Секреты GitHub

Добавьте Secrets:

- BOT_TOKEN — токен Telegram-бота
- ALIEXPRESS_API — ключ AliExpress API
- CHANNEL_ID — ID канала или чата (например @mychannel)

## Запуск

GitHub Actions запускает бота каждые 6 часов автоматически.
